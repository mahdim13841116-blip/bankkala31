# B2B Marketplace Platform

This project is a full-stack B2B (Business-to-Business) marketplace application connecting suppliers with store managers.

## Overview
The platform streamlines B2B operations including product catalog management, order processing, and financial settlements between suppliers and retailers.

## Architecture
- **Frontend**: React (SPA), Tailwind CSS.
- **Backend**: Express API server.
- **Database**: SQLite with Prisma ORM.
